muParser Archive
----------------

This directory contains old muParser packages. There are two different 
packages of muParser available. Since the project started on CodeProject 
there is a package for windows which requires VisualStudio and contains
a C# wrapper for .NET and there is generic package at SourceForge which 
should build on a variety of platforms (including windows). Both packages
contain the same source code but their build system is different.
